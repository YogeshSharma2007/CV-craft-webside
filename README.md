CV Craft is a full-stack web application designed to help users build and create professional, visually engaging CVs online with ease. The platform combines a modern, interactive frontend with a robust backend to deliver a smooth and dynamic CV-building experience.
The frontend is built using React, providing a fast, component-driven, and responsive user interface. To make the experience more engaging and visually distinctive, the app integrates Three.js, enabling interactive 3D elements and animations that give the platform a modern, creative edge — setting it apart from typical static CV builders.
On the backend, the application uses Node.js to handle server-side logic, API requests, and communication between the frontend and the database. This ensures efficient processing of user data, CV templates, and file uploads.
For data storage, the project uses Python with SQLite3 as the database layer, offering a lightweight, reliable, and easy-to-manage solution for storing user information, CV content, and related records without the overhead of a larger database system.
Key Features

Build and customize professional CVs through an intuitive interface
Interactive, visually appealing design powered by Three.js
Fast and responsive frontend built with React
Reliable backend processing with Node.js
Lightweight and efficient data storage using SQLite3
